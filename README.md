# Future-Library-Policy

1) 표현 유사도 (겉모양 변화)

문자 유사도 0.28 → 매우 낮음

변화의 대부분이 치환(재작성)
→ 문장 상당 부분이 새로 쓰였다.

2) 의미 유사도 (내용 연속성)

전체 유사도 0.18 → 낮음

초본 1문단 → 최종본 6문단

거의 그대로 유지된 문단 없음
→ 내용이 상당 부분 재구성됨.

3) 구조 변화도

초본·최종본 모두 제목 구조 없음
→ 이 지표는 사실상 비적용.

4) 정보 확장·축소

전체 토큰 감소(−32) → 소폭 축소

인용 표현 증가(+4) → 정의·강조가 늘어난 방향
